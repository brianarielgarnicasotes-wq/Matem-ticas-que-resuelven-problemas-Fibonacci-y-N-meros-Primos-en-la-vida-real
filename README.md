# Matemticas-que-resuelven-problemas-Fibonacci-y-Numeros-Primos-en-la-vida-real
Este proyecto muestra aplicaciones de las matemáticas en la vida real
 Contenido
- Fibonacci en el crecimiento de población 
- Números primos en la seguridad
Todo esto se realizara con HTML,Java script y CSS
En el crecimiende de poblacion se introduce  N numeros de meses y se mostrara el incremento de poblacion.
En primos en la seguridad se realizara lo siguiente :
-Cual quier numero no primo dara: "No primo "
-Y si es primo mostrara lo siguiente: Es primo (clave segura)
Estos dos ejercicios demuestran que la matematica resuelve problemas en la vida real cumpliendo el proposito de la pagina.
